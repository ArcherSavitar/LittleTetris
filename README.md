# LittleTetris 游戏

这是一个使用HTML、CSS和JavaScript实现的经典俄罗斯方块游戏。

## 特性

- 经典的俄罗斯方块游戏玩法
- 7种不同形状的方块 (I, O, T, S, Z, J, L)
- 方块旋转和移动控制
- 行消除和计分系统
- 难度等级随时间增加
- 下一个方块预览
- 游戏暂停功能
- 碰撞检测和游戏结束判断

## 控制方式

- 左/右箭头键: 移动方块
- 上箭头键: 旋转方块
- 下箭头键: 软降 (加速下降)
- 空格键: 硬降 (立即落到最底部)
- P键: 暂停/继续游戏

## 技术栈

- HTML5 Canvas 用于渲染游戏画面
- CSS 用于样式设计
- JavaScript ES6+ 用于游戏逻辑实现

- 4 行 (Tetris): 800 分 × 等级
